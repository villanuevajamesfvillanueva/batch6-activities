TicTacToe v1.1.5

v1.1.5 22.02.21 What's new?


  • Improved move history functionality. Bug: When history is accessed and previous and next button are used, the index changes depending on which item in the history the player is accessing. But, the last know value of index gets carried over the next round when the history is reenabled. It then gets updated immediately to its correct values when Previous and Next buttons are pressed.
  • Custom cursor implemented. Cursor changes depending on who's turn when hovered over the game board.


v1.1.2 22.02.21 updates:


  • Fixed proper winning cell highlighting.
  • Times where the game ends with two winning patterns can now be recognized.


v1.1.0 21.02.21 updates:


  • Added proper player interface.
  • Added scoring system.
  • Fixed board reset button.
  • Improved move history functionality. History can now be accessed using Previous and Next buttons but history index being used increases proportionally to the value of the current round.


v1.0.0 Known bugs features:


  • Move history in chaos.
  • Previous and Next button not behaving accordingly.
  • Reset button not properly resetting the board. Needs to reload the page everytime.
  • Winning cells sometimes fail to highlight.
  • Despite these, game is playable overall.
